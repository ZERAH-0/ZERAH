
ZERAH 포트폴리오 사이트 사용법

1. index.html 파일을 열면 전체 사이트 구조를 볼 수 있습니다.
2. 글을 수정하고 싶다면 <!-- -->로 표시된 부분을 찾아 내용을 바꿔주세요.
3. 이미지 파일은 /images 폴더에 들어있습니다. 교체하고 싶을 경우 동일한 이름으로 덮어쓰기 하세요.
4. GitHub Pages에 업로드하려면:
   - GitHub에 새 repository 생성 (예: zerah.github.io)
   - 이 zip 파일을 풀고, 모든 파일을 repo에 업로드
   - Settings → Pages → 소스(Source)를 main branch로 설정
   - 완료! https://zerah.github.io 에서 사이트가 열립니다.
