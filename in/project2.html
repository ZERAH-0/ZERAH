Deep Learning for Computer Graphics


🦖 Deep Learning for Computer Graphics

  기간: 2023.09 - 2023.12
  역할: 실험 설계 / 모델 구현 / 하이퍼파라미터 튜닝 / 결과 분석



  📌 연구 개요 (Overview)
  이 프로젝트는 컴퓨터 그래픽스와 딥러닝의 융합 실험으로,
(1) PyTorch3D 기반 3D Mesh 최적화 실험과
(2) 강화학습(DQN)을 활용한 물리 시뮬레이션 제어 학습
  두 축으로 이루어졌다.
  두 연구 모두 그래픽스 모델을 ‘데이터’가 아닌 ‘학습 가능한 물리 구조물’로 다루며,   AI가 시각적·물리적 피드백을 학습하는 시스템을 탐구하는 것을 목표로 했다.



  📌 연구1) Mesh Pitting Problem
   주제: 3D Mesh Optimization using PyTorch3D

  PyTorch3D의 Mesh Deformation 기능을 활용하여, 가늘고 유연한 3D 구조체의 형상 안정화와 세부 보존을 위한 손실 함수 조합을 실험했다. Chamfer Loss, Edge Loss, Normal Consistency, Laplacian Smoothing을 조합하고 가중치 및 옵티마이저(Adam / Momentum)를 비교하면서 손실 변화와 시각적 품질의 상관성을 분석하였다.

   주요 결과:
   • Uniform Laplacian 방식이 Cotangent보다 안정적인 형태 유지에 효과적임을 확인.
   • Momentum Optimizer가 Adam보다 Cat 모델의 형상 왜곡을 덜 유발.

  의의:
3D 모델의 미세한 구조 보존을 위해 손실 가중 조합의 중요성을 실증적으로 검증했으며, 그래픽스 실험에 물리적 제어 감각을 도입한 시도라는 점에서 의미가 있다.


  📌 연구2) DQN Lunar Lander
   주제: Reinforcement Learning for Physics-based Control

  OpenAI Gym 환경의 Lunar Lander-v2를 기반으로 DQN(Deep Q-Network)을 직접 구현하고, 행동 정책 학습을 통해 착륙 성공률을 높이는 실험을 수행하였다. 경험 리플레이(Experience Replay)와 타깃 네트워크(Target Network)를 병행하며 학습 안정성을 확보했고, 파라미터 변화가 에이전트의 의사결정에 미치는 영향을 시각화했다.

   실험 결과:
   • 초기 단계에서는 탐험(Exploration)이 우세, 500 에피소드 이후 수렴 경향.
   • ε-Decay를 조정하여 학습 효율 개선.
   • α(학습률)와 γ(할인율) 튜닝을 통해 Landing Precision 86% 달성.
   • 시각화된 궤적을 통해 에이전트가 ‘정상 착륙’ 행동을 안정적으로 학습했음을 확인.

  의의:
DQN의 기본 원리를 직접 구현·검증하며, 딥러닝 모델이 단순 시각적 데이터뿐 아니라 물리적 결과(속도, 각도, 충돌 등) 를 학습할 수 있다는 가능성을 확인했다. 이는 향후 시뮬레이션 환경에서의 자율적 AI 행동 제어 연구로 확장 가능한 기반이 되었다.


  📌 종합적 인사이트 (Insights)
   • 그래픽스와 물리, AI를 결합한 실험을 통해 시각적 모델이 ‘학습 가능한 물리 시스템’이 될 수 있음을 체감.
   • Loss Function 조합과 RL Policy의 튜닝 모두 "AI가 환경 피드백을 해석하고 반응하는 방식"을 다룬다는 공통점을 가짐.
   • 이를 통해 AI 기반 실시간 그래픽스 제어의 기초를 실험적으로 구축.



 📌 핵심 키워드 (Core Keywords)
