AI와 끝말잇기


🦖 AI와 끝말잇기 - Unity + OpenAI + Azure TTS

  기간: 2025.09 - 2025.10
  역할: 단독 개발 (AI 로직 · UI · 음성 통합)
  엔진: Unity 2022 LTS
  기술 스택: OpenAI API, Azure Speech TTS, C#, JSON, Unity UI
  플랫폼: PC / Web



  📌 프로젝트 개요 (Overview)
  ‘AI와 끝말잇기’는 인공지능과 사람이 단어를 주고받으며 경쟁하는 대화형 인터랙티브 게임이다. OpenAI의 언어 모델을 활용해 자연어 규칙 기반의 끝말잇기 로직을 구성하고, Azure TTS를 연동해 AI가 실제 목소리로 단어를 발음하도록 구현했다. 단순한 문자 입력 게임을 넘어, AI의 말투·표정·감정이 반영되는 시각적·청각적 인터랙션을 설계했다.



  📌 개발 방식 (Development)
  • AI 로직    - OpenAI GPT API로 끝말잇기 규칙을 정의한 프롬프트 설계 (금칙어·중복 방지 포함).
   - AI가 이전 단어의 마지막 글자를 기준으로 단어를 생성하도록 유도.
   - 잘못된 단어나 중복 시 자동 재시도 루프 처리.

  • 음성 합성 및 출력    - Azure Speech TTS(한국어 ‘서현’)을 이용해 AI의 응답을 실시간 음성으로 출력.
   -텍스트 버블과 음성 타이밍을 동기화하여 자연스러운 대화 흐름 구현.

  • UI/UX 구조    - 플레이어(노랑)와 AI(하늘색)의 양방향 말풍선 UI 구현.
   - 턴 전환 시 말풍선이 자연스럽게 등장·퇴장되는 애니메이션 적용.
   - 캐릭터 ‘Wordy’의 표정이 AI의 감정 상태에 따라 변하도록 애니메이션 매핑.

  • 턴 및 데이터 관리    - TurnManager에서 사용자 입력 → AI 응답 → 결과 검증 → 점수 업데이트까지 일괄 제어.
   - 대화 이력은 JSON 형태로 저장되어, 재시작 시 복원 가능.



  📌 주요 특징 (Features)
  • 대화형 인터랙션: AI가 음성·표정·말풍선으로 실시간 반응하며 사람처럼 대화하는 구조 구현.
  • 감정 기반 반응: 단어 난이도·턴 상황에 따라 다른 어조(놀람·고민·장난 등)로 음성 표현.
  • 실시간 음성 출력: AI의 발화가 곧바로 출력되며, 대화 텍스트와 정확히 동기화됨.
  • AI 메모리 시뮬레이션: 이전 대화 기록 기반으로 중복 단어 방지 및 문맥 유지 기능 구현.
  • UI 애니메이션: 대화 버블 생성·사라짐 타이밍을 오디오 재생 타이밍과 일치시켜 몰입감 강화.



  📌 성과 & 인사이트 (Outcome & Insights)
  • OpenAI API와 Azure TTS를 실시간으로 연동하며 AI 대화·음성 통합 파이프라인 완성.
  • 프롬프트 설계, 비동기 통신, 예외 처리 등 AI 인터랙션 설계의 핵심 기술 습득.
  • 시각·청각·언어를 결합한 감정 기반 대화형 UX를 구현하면서 AI 캐릭터 연출에 대한 통찰을 얻음.
  • 해당 구조를 확장해 이후 프로젝트(“감정형 캐릭터 시뮬레이션”)의 기반으로 발전시킴.



 📌 핵심 기술 (Core Stack)

   Unity / C# / OpenAI API / Azure TTS / Interactive Game / Chat UI / Emotion-driven AI / Async System / JSON Data / Real-time Dialogue
